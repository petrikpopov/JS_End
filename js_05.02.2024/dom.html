Document Object Model (DOM)

DOM - концепція, згідно з якою всі html елементи відображаются на обьекти (ООП) JS. Причому це відображення двобічна: змінная и html відбивається на структурі обьектів, а змінная структури обьектів - на html. Іншими словами , немає різніці як маніпулювати сторінками - через вплив на html чи шляхом змінни обьектів (та іх колекцій).
