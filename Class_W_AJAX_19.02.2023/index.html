AJAX

AJAX - Asynchronous JavaScript And XML - технологія ахенхроного обміну даними між клієнтом (JS) та сервером(поза браузером). Історично передбачався обмін в XML форматі , більше пошірений JSON , але назва технологіі зберігається.

Запит данних ініціюється (запускається) методом fetch Обовьязковий параметр - URL(адреса) данних. Оскільки запит асинхронний, його слід очікувати (await) або задавати функцію продовження .then(r => {}), яка одержує відповідь (r - response) від запиту. З відповіді можна дізнатись статус-код , заголовки та ініціювати(запустити) процедуру видобутку тіла відповіді. Тіло можно видобувати в одному з форматів .text() - як текст, JSON - як JSON, .blob(), .arrayBuffer() - як бінарні данні.
