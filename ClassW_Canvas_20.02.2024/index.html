Canvas

Полотно , засоби рисувань

Основу для складних забражень дає елемент <canvas> - полотно. З урахунку особливасті DOM - перезбирати весь документ після довільноі зміни - полотна надають покращені можливості створення динамічного контенту.
Лініі. Ідея рисування ліній полягає у комбінаціі двох операцій - переміщення "каретки" без сліду, та перемішення зі слідом. За замовченням розмір полотна 300 на 150 пікселів . При цьому важлило розуміти, що є розміри, а є роздатність, кщо стилями змінити розмір полотна , то його роздільна здатність не змінится, при збільшенні проявиться піусельна структура.

Курсы валют

Кешування(збереження) данних

  • Зменшення трафіку
  • Автономніть(можливість роботи без інтернету)
  • Збереження налаштувань

У JS у браузері є ряд обмежень зокрема доступ да файловой системи. Для задач збереження данных браузен надаэ декилька засобів доступних для перегляду у вкладці Аpplication інструменів розробника(консоль):

  • Local Storage - постійне сховише. Тримає значення що доступне усім вкладкам у яких відкрито дану сторінку. Відновлює значення після закриття вкладни і навідь браузера.
  • Session Storage - тимчасове сховище. Значення доступне лише данній вкладці. Закриття вкладки руйную значення. При оновлені сторінки - значення зберігаеється.(закрити , відкрити)
  • Index DB - для великих данних типу "Ключ-значення".
